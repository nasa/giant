giant.scripts¶

This subpackage provides a number of command line scripts to do various tasks in GIANT.

The content and capabilities of these scripts vary widely so see the following summary and detailed documentation for more details.

Available scripts¶

Scripts

build_catalogue

Build a giant star catalogue file from the UCAC4 star catalogue.

generate_sample_data

Generate sample images and camera meta-data for doing the examples included throughout the documentation.

ingest_shape

Ingest a tessellated shape model and store it in a GIANT KDTree for use in GIANT relative OpNav processes.

merge_cameras

Merge camera files together into a single file.

mosaic_images

Create a mosaic image from GIANT cameras.

shape_stats

Compute statistics on a shape contained in a giant kdtree saved to a pickle file.

spc_to_results

Convert SPC Autoregister results into a GIANT results pickle file format.

spc_to_feature_catalogue

This script provides a utility for converting a SPC set of Maplets into a GIANT feature catalogue for use with GIANT Surface Feature Navigation (SFN).

tile_shape

Tile a shape model into SurfaceFeatures in a FeatureCatalogue.
