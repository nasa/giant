merge_cameras¶

giant.scripts.merge_cameras:

Merge camera files together into a single file.

Camera files are assumed to be stored in dill or pickle files and are store in a dill file. The camera files should contain the same Camera class with different images. This script does not enforce this though so be careful as you may experience unexpected results if you try to merge files containing different Camera classes.

Warning

This script loads/saves some results from/to python pickle files. Pickle files can be used to execute arbitrary code, so you should never open one from an untrusted source.

Usage¶

Merge images between two camera files. NOTE: The cameras must be the same as this will only merge the images.

usage: merge_cameras [-h] [-o OUTPUT] cameras [cameras ...]

Positional Arguments¶

cameras

The camera files to merge

Named Arguments¶

-o, --output

The file to store the merged camera object

Default: './camera.dill'

WARNING: This script loads/saves some results from/to python pickle files. Pickle files can be used to execute arbitrary code, so you should never open one from an untrusted source.
