StellarOpNav.update_point_of_interest_finder¶

giant.stellar_opnav.stellar_class:

StellarOpNav.update_point_of_interest_finder(point_of_interest_finder_update=None)[source]¶

This method updates the attributes of the point_of_interest_finder attribute.

This method also updates all of the elements of process_stars to True.

Parameters:

point_of_interest_finder_update (PointOfInterestFinderOptions | None) – An instance of PointOfInterestFinderOptions containing the new settings
