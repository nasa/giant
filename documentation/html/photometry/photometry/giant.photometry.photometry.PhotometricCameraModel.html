giant.photometry.photometry.PhotometricCameraModel¶

class giant.photometry.photometry.PhotometricCameraModel(gain, transfer_time, standard_mag, bin_mode, resolved_rows_threshold, dn_readnoise, dn_rate_standard, dark_current, psf_factor, camera_model, name=None, max_exposure=200000.0, **kwargs)[source]¶

Bases: object

This class is used to set parameters for a Photometric Camera

Required inputs are used to estimate and plan photometric targeting with the camera. Additional parameters can be set to define the camera model

Parameters:

  • gain (float) – A float representing the camera gain in dn

  • transfer_time (float) – A float representing the camera transfer time in seconds

  • standard_mag (float) – standard magnitude

  • bin_mode (int) – bin mode of camera

  • coadd_num – coadded images

  • resolved_rows_threshold (int) – number of pixel rows to define a target as resolved

  • dn_readnoise (float) – background dn from the camera noise

  • dark_current (float) – a float representing the dark current

  • dn_rate_standard (float) – DN/s for I/F = 1 at 1AU

  • psf_factor (float) – factor for the point spread function

  • camera_model (CameraModel) – giant camera model

  • name (str | None) – a name for the camera model

  • max_exposure (float) – the maximum exposure time the camera can manage in seconds
