giant.photometry.photometry.Photometry¶

class giant.photometry.photometry.Photometry(scene, photometric_camera_model=None, scattered_light=None)[source]¶

Bases: object

This is most useful when you would like to determine the required photometry inputs for opnavs given a certain date. In addition to the Scene methods, this class contains calculations for iof, dnrate, snr, etc. This class can also determine target pariapse given an estimated time and move the scene to that time, as well as determine if the target is resolved during the scene.

This is also useful because it can calculate the apparent location of objects in the scene (if they have the position_function and orientation_function attributes defined) while applying corrections for light time and stellar aberration. This can be done for all objects in a scene using method update() or for individual object using calculate_apparent_position(). Any objects that do not define the mentioned attributes will likely not be placed correctly in the scene when using these methods and thus warnings will be printed.

Parameters:

  • scene (Scene) – Scene containing at least one target and lighting object

  • photometric_camera_model (PhotometricCameraModel | None) – PhotometricCameraModel object

  • scattered_light (ScatteredLight | None) – scatteredLight object used to define how scattered light is handled. If none provided, scattered light is not taken into account

scene¶

Scene containing at least one target and lighting object

photometric_camera_model¶

Photometric Camera model object used to define constants of the camera

scattered_light¶

scatteredLight object used to define how scattered light is handled

Summary of Methods

sea_angle

This method computes the solar elongation angle defined as the angle between the light_obj, the observer and the target at target_index.

resolved

This method determines if the the target at target_index is resolved by the camera.

iof

This method determines the Luminosity I/F of the the target at target_index given the position during the scene.

apparent_mag

This method determines the apparent magnitude of the the target at target_index given the position during the scene.

dnrate

This method determines the DN rate of the the target at target_index and determines which model to used based on if the requested target is resolved

snr

This method determines the Signal to Noise Ratio of the the target at target_index based on the input exposure time.
