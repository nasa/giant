Photometry.apparent_mag¶

giant.photometry.photometry:

Photometry.apparent_mag(target_index, magnitude_model)[source]¶

This method determines the apparent magnitude of the the target at target_index given the position during the scene.

The magnitude is based on an input PhaseMagnitudeModel Class that calculates the magnitude of the target at target_index based on the current phase angle at the scene, in radians.

It is up to the user to determine the proper magnitude model to used based on if the target is resolved or unresolved.

Parameters:

  • target_index (int) – the index into the target_objs list for which to compute the phase angle for

  • magnitude_model (PhaseMagnitudeModel) – magnitude_model used to determine apparent magnitude of target

Returns:

apparent magnitude of target.

Return type:

float
