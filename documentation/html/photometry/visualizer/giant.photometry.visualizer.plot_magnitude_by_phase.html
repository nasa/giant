plot_magnitude_by_phase¶

giant.photometry.visualizer:

giant.photometry.visualizer.plot_magnitude_by_phase(magnitude_model, scene, scene_object)[source]¶

Plot a Phase Magnitude Model. Resulting figure will plot phase angle (deg) vs. magnitude with an x representing the current geometry.

Parameters:

  • magnitude_model (PhaseMagnitudeModel) – PhaseMagnitudeModel instance for plotting

  • scene (Scene) – Scene representing the current geometry

  • scene_object (SceneObject) – SceneObject or Scene object to find magnitude

Returns:

matplotlib figure and axes objects containing plot

Return type:

tuple[Figure, Axes]
