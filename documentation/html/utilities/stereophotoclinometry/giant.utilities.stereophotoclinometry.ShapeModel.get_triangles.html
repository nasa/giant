ShapeModel.get_triangles¶

giant.utilities.stereophotoclinometry:

ShapeModel.get_triangles(triangle_precision='64')[source]¶

This method returns GIANT triangle objects from the maplet data to be used in the GIANT ray tracer

Parameters:

triangle_precision (Literal['64', '32']) – A flag specifying whether to return 64 bit or 32 bit triangles

Returns:

The GIANT triangles

Return type:

Triangle64 | Triangle32
