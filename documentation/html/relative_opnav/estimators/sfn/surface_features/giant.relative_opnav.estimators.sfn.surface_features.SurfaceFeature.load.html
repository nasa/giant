SurfaceFeature.load¶

giant.relative_opnav.estimators.sfn.surface_features:

SurfaceFeature.load()[source]¶

This method loads the shape information from disk.

Typically a user will not use this method directly as it is automatically called by the shape and bounding_box properties when required.

Raises:

  • ValueError – If the shape file information is not available

  • IOError – If the shape file is not available
