usnognc¶

This module provides a class for interfacing with the USNO GNC (Guidance Navigation and Control) star catalog.

The GNC catalog provides high-precision astrometric data based on GAIA DR3, Hipparcos, and USNO’s bright star catalog for navigation applications.
