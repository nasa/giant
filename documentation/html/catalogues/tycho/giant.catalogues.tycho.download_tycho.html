download_tycho¶

giant.catalogues.tycho:

giant.catalogues.tycho.download_tycho(target_directory)[source]¶

This function downloads the Tycho2 catalogue from vizier to the target directory.

This is done over ftp. It requires an active internet connection that can connect to cdsarc.u-strasbg.fr

Warning

This download will take a long time and use up approximately 500 MB of space.

Warning

This download has no way to verify the integrity of the files because no hash is provided. While the vizier service is trusted, use this function at your own risk

Parameters

target_directory (pathlib.Path) – the directory to save the Tycho catalogue to
